Why should you ensure that the script element that links to the jQuery library file comes before any other script element for a .js file that includes jQuery code?

The order ensures that browsers parse the contents of the jquery library before attempting to interpret the Jquery statements in script.js. Otherwise the Jquery syntax in script.js could cause syntax errors if browsers encounter it before parsing the Jquery library.

++++++++++++
Why should you avoid including jQuery in a project if you don't need it?

JQuery can slow the time required for your app to load because users have to download the JQuery library.

What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.

A content delivery network is a large distributed system of proxy servers deployed in multiple data centers via the Internet.
CDNs are optimized for quick delivery of files across the globe, when linking to a file on CDN it provides an improved user experience. A CDN can distribute the load, save bandwidth, boost performance and reduce your existing hosting costs.
